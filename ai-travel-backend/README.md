# AI Travel Backend

AI-Based Smart Travel Experience & Management System.

## Project Structure

```text
ai-travel-backend/
│
├── src/
│   ├── config/          # Database & configuration
│   ├── modules/         # Feature-based modular logic
│   ├── shared/          # Common middleware, utils, and constants
│   ├── seeders/         # Data seeding scripts
│   ├── routes/          # API entry point
│   ├── app.js           # Express app setup
│   └── server.js        # Server entry point
│
├── .env                 # Environment variables
├── package.json         # Dependencies & scripts
└── README.md            # Project documentation
```

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `.env`.

3. Start the server:
   ```bash
   npm run dev
   ```
